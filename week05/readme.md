E-Commerce Analytics Dashboard

A professional analytics dashboard built using Flask, Python, Pandas, and Plotly to analyze e-commerce data. It provides interactive dashboards for Sales, Customers, Products, Sellers, Delivery, Payments, and Reviews, helping businesses gain actionable insights through visualizations and key performance indicators (KPIs).

Features

Interactive Dashboard
Sales Analytics
Customer Analytics
Product Analytics
Seller Analytics
Delivery Analytics
Payment & Review Analytics
Dataset Explorer with Search, Filter, Sort, and Pagination
